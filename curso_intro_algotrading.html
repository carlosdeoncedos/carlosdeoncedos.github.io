Sobre el curso

"Introducción Algotrading" es un curso introductorio a la programación financiera y a Python, el lenguaje computacional número uno para el análsisi financiero y Machine Learning. Al terminar el curso serás capaz de desarrollar, implementar y evaluar el desempeño de algoritmos financieros para la toma de desiciones de compra o venta de instrumentos, transformar estrategias de inversión en algoritmos computacioneles, crear funciones, etc.

Temario

  • Qué es Python
  • Diferencia entre Python y Jupyter
  • Tipos de datos y sus estructuras (Texto, números, variables, listas, tuplas y diccionarios)
  • Lógica computacional (if, for loop, while)

  • Qué es una librería en Python
  • Introducción a las librerías
    • Numpy
    • Matplotlib
    • Pandas

Para cada algoritmo:
  • Explicación de la teoría del análisis
  • Codificación del algoritmo
  • Evaluación del algoritmo

  • Breve introducción a la teoría de los portafolios de inversión
  • Generar portafolio cuantitativo
